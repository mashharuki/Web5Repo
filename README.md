# Web5Repo
Web5SDKを調査・学習するためのリポジトリです。

## WEB5 SDKとは

TBDが開発するVC・DID周りの機能を提供するライブラリ。

## Decentralized Web Nodes(DWNs)とは

DID関連のデータを保管するデータストレージ(TBDが定義したもの)

## Web5 内での Protocols

共通してデータをやり取りするためにアプリごとに定義する企画

## VCに必要な要素

- Type
- Issuer
- Subject
- expirationDate
- data
